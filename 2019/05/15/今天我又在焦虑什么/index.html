今天我又在焦虑什么
发布于 2019-05-15 / 更新于 2019-05-15 / 姐告小城 / 热度 0 / 喜欢 0

时间过得好快啊，上一次写日记好像还是一个月前，这一个月又发生了太多事情

首先说一下，为什么突然更新？原因还是受到了闺蜜的刺激，发现她生长得好快，思想，学习都成长得太快，我像一个小绿苗瑟瑟发抖起来。于是，赶紧记录下这个事情，复盘下我的生活。

说下这一个月发生了什么大事？、

打了右耳耳钉，快二十天了，还没愈合。我的体质原因，代谢能力好像不高，所以愈合较慢。

喜欢上了周震南小哥哥，对，我开始追星了，阿西吧！这真的是耽搁时间的事情，但快乐啊！为什么粉上这个小哥哥？他比我小了整整6岁

就创作者来说，他很有野心，作品有往深入走，想表达很多他想要表达的东西，愿意去影响大众。我喜欢这样的创作人。

价值观上，他给了我很大的冲击感。关于艺术，关于职业，关于当下的社会，他想表达的价值观是和我赞同的且欣赏的，审美也是重合的，哥们儿也很喜欢花花绿绿色彩鲜艳的东西，私下就很喜欢宽松舒适简单。

另外，给了我很多激情的东西。永远不是年龄给予人意义，而是人去赋予年龄意义。

两年以来，我蛮困顿的，总感觉很多事情我不能做了，我不能跳舞，我不能追求我以前的职业，我不能下腰太多，膝盖会疼，我要更加爱惜自己，我知道人跟人不一样，有的人选择冒险，有的人选择舒适，我从而走不出我的舒适圈。

关键是我不是一个喜爱舒适的人。

可能有一点点喜欢，但我不是一个长久归于舒适的人，舒适反而使我痛苦。

对，爱豆写了一首歌，叫做镜子。镜子是尼采说过的一句话才创作出来的，人的一生就是一面镜子，而我们要做的事情就是从镜子里面认清自己。

知道这一点，我就开始了我不断的冒险，我也开始重新锻炼自己。现在很开心，下班还能嗨一个小时的舞蹈，遗憾是时间精力不够，不能看书了。

去看了即兴现代舞，讲述自我的主题，评书专场，这两个活动给我的感受是，创作者，必须坚持自我，也必须加入别人没有的东西，如果一味地跟随市场同质化，只有被大浪淘沙的大浪淹没，想要别人对你的东西感兴趣，一定要做到两个字，创和新。

一个是三十多岁的评书先生，一个是四十岁的舞者，在这个行业里，大家都明白一件事情，我们服务的对象，观众，永远是年轻的。垂老，意味着被时代，市场，艺术的审美变化所淘汰。同时，也意味着，你的审美随着经验阅历不断加深，不断走上顶层，从而包容更多的审美。

两个方面。

再来是五一去看了三个展览，穆夏的展览，标题定位欧洲新艺术运动先锋，OH MAY GAD，我完全没理解到新艺术的概念。穆夏是做设计，大多数展图是设计海报和各种设计稿，可以见的他的作品是不蕴含自我的，少许有他向表达的东西。

总体上是非常美的。

还有一个是贝里尼家族的收藏特展，蒙娜丽莎的微笑和文艺复兴时期的雕塑，工艺品，绘画，这个真的太棒了！我终于get到了观看之道。

虽然并不是约翰告诉的那样，第三点，艺术感受力。

在我看来，原画与CG出图的最大区别就在于现场的透视问题吧。

这我是在听评书的那天感受到的，去的那一晚，先生讲薛仁贵东征时，背后有一副镇店的山水工笔，当时觉得像雕刻的一般，那个透视看上去就是如同浮雕。

我反复看了几遍，还是确定这是一幅画，因为它的阴影不随头顶上的光改变，但某个角度看上去，真的很像浮雕，出来3D透视的效果。

绝了。

在贝里尼展览上也遇到了很多这样的画，这大概就是现场看展的魅力，无法从再版身上得到的魅力。

接下来就是最愧疚的一件事，那就是我和苏苏走远了。

唉，提起来真是伤心，但内心又无动于衷。我真是个垃圾，所以我的成长不快，我没有把事情做好，我就是在焦虑这个。

太忙了，我放弃了我的创作，半月时，我还会和苏苏维持关系，互相讨论很多东西，后面就慢慢淡了。

你要知道，两个人走近对方，了解彼此，迁就彼此，愿意去感受彼此所感受的事情，是多么难得的事情，我们花了近一年的时间才成为很了解对方的朋友，却在半月的时间，消弭殆尽。

苏苏还会坚持给我发消息，告诉我很多事情，我每天回家，就没有太多心情了。

早前说过，关系是个很微妙的东西，你想抓紧，别人却没有想靠近的心，这能怎么办呢？这是最无奈的事情。苏苏偶尔会问我，为什么不理她？

内心真愧疚，偏偏我也不敢回答。归根结底我的性格就是这样，我能够同理到别人的心，我却无法做到那些事情。

总之，这段关系我也不知道怎么处理，但总不可能就这么失去了吧。

就比如肖肖过生日，我说希望给她做个庆贺生日的视频剪辑，是我们两年追过的男人们！但每天时间忙到，也没有做，就感觉时间不够用。答应了她的事情，又没有完成，陷入焦虑和自责。

我又不愿意消耗更多的精力，牺牲自己，来把事情做好。

这样是不对的。

！！！

还有就是我马上又要折腾了，这周周六约了个纹身师傅，马上要去纹身了，也不知道会不会排异，排异好像会给身体造成很重的负担。

pink tiger

纹身我一直想纹，却总是说，工作稳定了再说，拖着，拖着，感觉这件事就不会做了。看看，我就是这样一个没有勇气的人，又有一点讨好人格，不够果断，不够坚持，总是找借口拖延，我说的最多的就是但是。

借口都是合理的，超出合理的范围，才是我要去达到的，世界上的人都能做到合理的事情，但不是每个人都能做到给别人惊喜的事情。

我希望我能做出很多惊喜的事情。

pink tiger就是我对自己的美好祝福。

粉色是希望我永远少女，拥有女性的美丽。tiger 是希望我自己勇敢一些，刚烈一点，再辣一点，做过了的事情，就不要去纠结，更加独立。

另外，学习上。

总感觉在不停地赶进度，向前冲，学习很多新的东西，但其实慢下来，对以前的知识进行复盘，深入，会有很多的帮助。还有一个点，我会把未做完的事情记录下来，每天堆着堆着，所以呢？

我需要空出一个时间去完成这些事情。

总结：焦虑自己成长得太慢，却是自己不愿意花时间去塑造自己

归根：还是我的问题

balabalabala,真的好啰嗦。我真是一毛病多的女人。
